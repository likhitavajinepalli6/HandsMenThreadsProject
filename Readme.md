## Overview
HandsMen Threads is a Salesforce-based CRM system developed during my SmartBridge Virtual Internship. The project streamlines customer, product, inventory, and order management while automating business workflows using Salesforce.

## Features
- Customer management
- Product management
- Inventory tracking
- Order management
- Role-based access control
- Custom objects and validation rules
- Salesforce Flow automation
- Apex Triggers
- Dashboards and Reports

## Tech Stack
- Salesforce CRM
- Apex
- SOQL
- Salesforce Flow Builder
- Lightning Experience
